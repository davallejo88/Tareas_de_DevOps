Punto 3 Documentación de Configuración

Instalación y Configuración de Apache

1. Instalación de Apache

apt-get install apache2

2. Iniciar el servicio de Apache2

sudo systemctl start apache2

3. Modificar el puerto por defecto (80 a 8088)

vi /etc/apache2/ports.conf

4. Proporcionar permisos sobre las rutas

sudo chmod 0777 /var/www/html/ sudo chmod 0755 /var/www/html/ sudo chmod 0644 /var/www/html/index.html
